# CareBridge SOS System

Hệ thống điều phối và dẫn đường y tế khẩn cấp (SOS), bao gồm hai thành phần chính:
1. **Frontend (Flutter):** Ứng dụng di động bản đồ sử dụng MapLibre GL và TrackAsia API.
2. **Backend (Java Spring Boot):** Máy chủ API quản lý dữ liệu không gian (Spatial Data) với PostGIS.

## Cấu trúc dự án
- `/frontflutter`: Chứa mã nguồn ứng dụng Mobile (Android/iOS).
- `/backjavaspring`: Chứa mã nguồn máy chủ (Backend).

Vui lòng xem file `README.md` bên trong từng thư mục để biết cách cài đặt và chạy chi tiết.
